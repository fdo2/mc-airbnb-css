## CSS Variables

We have defined some CSS variables to be used in style rules. Check them out at the top of the `main.css` file.

Variable definition:
```css
html {
  --spacing-x-large: 4rem;
}
```

How to use them:
```css
.my-class {
  margin: var(--spacing-x-large);
}
```
